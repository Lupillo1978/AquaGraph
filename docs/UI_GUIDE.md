# AD&M AquaControl

## Guía de Interfaz de Usuario

Versión: 1.0

---

# Filosofía

AD&M AquaControl debe permitir que el operador conozca el estado completo de la granja en pocos segundos.

Toda la interfaz debe privilegiar:

- claridad
- rapidez
- consistencia
- simplicidad

---

# Colores de estado

## Gris

Equipo creado pero aún sin configurar.

## Verde

Equipo en línea.

## Naranja

Equipo alimentando.

## Rojo

Sin comunicación.

## Morado

Mantenimiento.

## Azul

Equipo seleccionado.

---

# Estanques

Color normal:

Verde translúcido.

Color seleccionado:

Amarillo.

---

# Alimentadores

Cada alimentador mostrará:

- Estado
- Identificador
- Tooltip
- Menú contextual

---

# Mapa

El mapa es el elemento principal del sistema.

Todos los equipos deben colocarse y administrarse desde él.
